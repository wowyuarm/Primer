# Neon-Noir

> "All those moments will be lost in time, like tears in rain."

## Narrative Core

2019年的洛杉矶，永恒的雨夜，霓虹灯在黑暗中流血。Blade Runner 创造了赛博朋克的视觉圣经：高科技与低生活，人造光与自然暗，未来的废墟美学。

Blade-Runner 是一个暗色优先的设计系统，探索在黑暗中如何用光来引导、强调和创造情绪。

## Systemic Rules

- **Color**: 深色背景 (#0D0D0D)，霓虹色仅用于强调和交互
- **Lighting**: 光源即层级——最亮的元素最重要
- **Typography**: 单线几何字体，略带未来感的字间
- **Interaction**: 发光悬停效果，如霓虹灯被激活

## Signature Constraint

**光暗比至少 1:7** — 发光元素（文字、边框、按钮）与背景的对比度必须极高。中间调是被禁止的——要么是深暗，要么是发光。

这意味着：
- 无柔和的灰色渐变
- 无低对比度的"舒适"配色
- 界面如同在黑暗中用霓虹灯写字
