# Rams-60

> "Weniger, aber besser." — Less, but better.

## Narrative Core

1960年代的博朗设计室。迪特·拉姆斯的十原则不是口号，而是工业设计的圣经。每一个按钮、每一条线都有其功能性理由。

Rams-60 是对**工业产品设计**的数字化译介。它适合需要像博朗计算器一样精确、诚实、持久的工具型界面。

## Systemic Rules

- **Color**: 灰阶为主，博朗橙 (#E64A19) 作为唯一强调色
- **Grid**: 4px 基准单位，如计算器按钮般精密
- **Typography**: Helvetica Neue，Minor Third (1.2) 字体阶梯——理性、可预测
- **Interaction**: 0ms 过渡，即时响应，如物理按钮

## Signature Constraint

**零装饰属性** — 禁止 `box-shadow`, `border-radius`, `gradient`, `text-shadow`。

设计的力量必须完全来自布局、字体和色彩的精确运用，而非装饰性效果。

## 适用场景

- 仪表板与数据展示界面
- 专业级生产力工具
- 技术文档与知识库
- 任何需要“诚实而持久”的界面
