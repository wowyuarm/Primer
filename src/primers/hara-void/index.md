# Hara-Void

> "这是空。空即是可能性的容器。"

## Narrative Core

无印良品的设计哲学不是"极简"，而是**空**（Emptiness）。Kenya Hara 说，空不是"什么都没有"，而是"什么都可能发生"的状态。

当设计退到背景，用户的生活才能走到前景。Muji-Void 不追求引人注目，它追求**消失**——成为日常的一部分，不被察觉却不可或缺。

## Systemic Rules

- **Color**: 只用自然界存在的中性色调——米白、暖灰、原木棕
- **Typography**: 日本美学中的"间"（Ma），大量留白，宽松行高 (1.8-2.2)
- **Scale**: Major Second (1.125) — 最温和的字体阶梯，变化几乎不可察觉
- **Interaction**: 慢速过渡 (400ms)，动作如呼吸般自然

## Signature Constraint

**禁止品牌色** — 不允许使用任何高饱和度、引人注目的颜色。一切色彩必须像在自然中能找到的那样：树皮、石头、棉麻、纸张。

这意味着：
- 无鲜艳的蓝、红、绿
- 无人造感的纯黑 (#000000)
- 无视觉冲击力的对比
